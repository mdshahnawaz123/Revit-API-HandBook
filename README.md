## ⚙️ How to Use

### Step 1 — Clone the Repository
Clone this repository to your local machine to access the handbook offline.
```bash
git clone https://github.com/mdshahnawaz123/Revit-API-HandBook.git
```

### Step 2 — Open the Handbook
Simply open `index.html` in any modern web browser to start exploring the books.
- **Chrome / Edge / Safari** are recommended for the best visual experience.
- No local server is strictly required, but using one (like `npx serve`) provides the smoothest navigation.
